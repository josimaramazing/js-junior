### Instalación del framework de pruebas

#### **Clonar el repositorio:**

```bash
git clone 
```

#### **Instalar y configurar Jest.**

```bash
npm init -y
npm i -D jest
npx jest --init
```

#### **Para la ejecución de los test**

```bash
    npm test
    npm test:watch
```
